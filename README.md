# Interview System - Deployment

## 🚀 Overview

This project is a microservices-based interview system built using Spring Boot and Docker.

## 🧩 Services

* Auth Service
* Quiz Service
* Question Service
* Student Service
* API Gateway
* PostgreSQL Database

## 🐳 Running the project

```bash
docker-compose up --build
```

## 🌐 Access

* API Gateway: http://localhost:8080

## 🛠 Tech Stack

* Java 17
* Spring Boot
* Docker & Docker Compose
* PostgreSQL
